# Ultimate Anime Quiz

An interactive, anime-themed quiz application that tests your knowledge across various popular anime series.

## Description

Ultimate Anime Quiz is a dynamic, engaging web application that challenges anime fans to test their knowledge of different anime universes. The quiz features questions from popular series like Naruto, One Piece, Attack on Titan, Demon Slayer, and Jujutsu Kaisen, with dynamically changing themes, sound effects, and backgrounds to create an immersive experience.

## Features

### 🧠 Quiz Functionality
- **Diverse Question Types:** Multiple-choice questions about anime characters, powers, scenes, and trivia
- **Dynamic Theming:** Background and styling changes based on the current anime question
- **Various Categories:** Questions from Naruto, One Piece, Attack on Titan, Demon Slayer, and Jujutsu Kaisen

### 🌈 UI & Visual Effects
- **Immersive Backgrounds:** Full-screen background images that change with each anime
- **Animated UI:** Colorful buttons with hover effects, glowing borders, and visual transitions
- **Custom Anime Themes:** Different fonts, colors, and styling for each anime series
- **Interactive Elements:** Click animations for all user interactions

### 🔊 Sound Effects
- **Anime-Style Audio:** Sound effects for all interactions
- **Contextual Sounds:** Different sounds for correct answers, wrong answers, question loading, and more

### 📱 Responsive & Accessible
- **Mobile-Friendly Design:** Fully responsive layout for all screen sizes
- **Accessibility Features:** Keyboard navigation support
- **Cross-Browser Compatible:** Works on all modern browsers

### 🏆 Score & Storage
- **Real-Time Score Tracking:** Progress bar and score counter
- **Personalized Results:** Anime-style verdict based on your performance
- **Local Storage:** Saves high scores and past attempts

### 🧩 Extra Features
- **Theme Toggle:** Button to manually switch between anime themes
- **Sound Controls:** Option to mute/unmute sound effects

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API for data persistence

## How to Run the App

1. Clone or download this repository
2. Open the `index.html` file in any modern web browser
3. Click "Start Quiz" to begin testing your anime knowledge!
